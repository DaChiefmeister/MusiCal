    • Frequently Asked Questions

    Q: What is MusiCal?

    Please refer to our About page for more detailed information about MusiCal.

    Q: Where does MusiCal get its event information from?

    Musical congregates information from across the web as well as user submitted events. For an explanation of how this works, visit our About page!

    Q: How do I set up an account?

    If you are using MusiCal for the first time, you can create an account through the “Login” link. You will be given an option to input your username and password or create a new account.

    Q: How do I search for and event?

    A search bar should be located on the top right side of your screen. If you are on a mobile device, look for a magnifier icon in the same area. You should be able to input any key words related to the events you are interested in.

    Q: How do I submit an event?

    To submit an event to musical, visit our Contact page and fill out the event submission form. You will receive a confirmation once your event has been successfully added onto the website.

    Q: How do I update an event I submitted?

    To make changes to an event, submit a Contact form and state any changes you’d like made to the event. You will receive a confirmation once the information has been received and the event should be updated shortly after.
